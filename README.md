# 🔐 Django Authentication API

A secure RESTful Authentication API built using **Django** and **Django REST Framework**. This project provides user authentication features including user registration, login, logout, profile management, and token-based authentication.

---

## 📌 Features

- ✅ User Registration
- ✅ User Login
- ✅ User Logout
- ✅ Token Authentication
- ✅ User Profile API
- ✅ Password Change
- ✅ Secure API Endpoints
- ✅ JSON Responses
- ✅ Postman Collection Included

---

## 🛠️ Technologies Used

- Python 
- Django
- Django REST Framework
- SQLite3
- Token Authentication
- Postman
